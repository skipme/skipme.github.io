Y-Translate Application (beta)

Some basics

Description

This application will help you in translation.
It accept texts, subtitles, books.
- Warning - in beta status (do backups of projects by using -exporting-)

Some terms

Project - object contains versions.
Version - object contains original and translated sentences.

Features

- Machine translation (online)
- Offline (your data stores only on your computer)
- Collaboration (commits translated sentences to origin point, publications, decentralised connections (no single point of failure), not implemented yet)
